# DKN
### if you want to run this code on the books repository you should uncomment the rows under # books in src/main.py and comment the rows under # news
### Required packages
The code has been tested running under Python 3.6.5, with the following packages installed (along with their dependencies):
- tensorflow-gpu == 1.4.0
- numpy == 1.14.5
- sklearn == 0.19.1
- pandas == 0.23.0
- gensim == 3.5.0


### Running the code
```
$ cd src
$ python main.py (note: use -h to check optional arguments)
```
